数学分析笔记

实数与序列

常见数集

下面给出一些集合的定义
$$
\N = {0, 1, 2, \ldots}\
\Z = {\ldots,-1,0,1,\ldots}
\
\Q = {\frac{p}{q}\vert p, q \in \Z, q\not = 0}
$$
对于四则运算封闭的是 $\Q, \R, \C$

Dedkind分割

先定义一个分割

$A, B$ 为有理数集合，满足：

  1. $A, B \not = \empty, \Q = A \cup B, A \cap B = \emptyset$
  2. $\forall x\in A, y \in B, x < y$

显然有四种情况

  1. $A$ 中有最大值，$B$ 中有最小值
  2. $A$ 中无最大值，$B$ 中有最小值
  3. $A$ 中有最大值，$B$ 中无最小值
  4. $A$ 中无最大值，$B$ 中无最小值

第一种情况不会出现，所以我们只讨论后面的三种。

Dedkind 分割就是 ${(A, B)是 \Q 的一个分割，且 A 中无最大值}$

每一个 Dedkind 分割唯一对应一个实数，无理数对应了第四种，有理数对应了第二种。

Dedkind 分割来定义无理数的运算

定义 $x = (A_1, B_1), y = (A_2, B_2)$ ，若 $A_1 \sub A_2$ 称 $x\le y$，若 $A_1 \sub A_2$ 且 $A_1 \not = A_2$ 称 $x < y$。

性质

  1. 反称性：$x\le y$ 则 $y\ge x$
  2. 传递性：$x\le y$ 且 $y \le z$，则 $x \le z$
  3. 完备性：$x, y\in \R$，则 $x\le y$ 或 $y\le x$

四则运算定义

$x + y = (A_3, B_3)$，其中 $A_3 = {r \in \Q\vert \exists r_1 \in A_1, \exists r_2 \in A_2, \text{s.t.} r = r_1 + r_2}$， $B_3 = Q/A_3$

负元 $-x = (A_3, B_3)$，其中 $A_3 = {r \in \Q\vert \exists r_1 \in B_1, r = -r_1, r_1 不是最小元}$， $B_3 = Q/A_3$

则减法定义为 $x - y = x + (-y)$

乘法：

$xy = (A_3, B_3)$

  1. $x\ge 0, y \ge 0, A_3 = {r \in \Q\vert r < 0 或 \exists r_1 \in A_1, \exists r_2 \in A_2, \text{s.t.} r_1 \ge 0, r_2 \ge 0, r = r_1r_2}$
  2. $x\ge 0, y < 0,xy = -(x(-y))$
  3. $x < 0, y < 0, xy = (-x)(-y)$

逆元 $A_3 = {r \in Q\vert r\le 0 或 \exists r_1 \in B_1 , r_1 不是 B_1最小元, r =\frac{1}{r_1}}, B = Q / A_3$

实数域范围内：

  1. 交换律
  2. 结合律
  3. 单位元
  4. 逆元
  5. 分配律

相容性条件

  1. 加法保序性 $x\le y$ 则 $x + z \le y + z$
  2. 乘法保序性 $x\le y, z \ge 0$, 则 $xz\le yz$

实数确界存在定理

定义 $X \sub \R,x\not = \empty$

  1. 若 $C \in \R, \text{s.t.} \forall x \in X, x \le C$ 则称 $C$ 是 $X$ 的一个上界，下界同理
  2. 若 $X$ 既有上界又有下界，那么称 $X$ 是有界集
  3. 称 $a$ 是 $X$ 的一个上确界，若 $\forall C$ 是 $X$ 的上界，$C \ge a$，记 $a = \sup X$ 上确界，下确界是 $\inf X$

定理：确界存在定理：有上界或者下界的非空子集，必有唯一的上确界或者下确界。
