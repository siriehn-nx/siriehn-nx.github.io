数学分析笔记

发表于 2023-09-18

数学分析笔记

实数与序列

常见数集

下面给出一些集合的定义 \[ \begin{aligned} \mathbb N &= \{0, 1, 2, \ldots\}\\ \mathbb Z &= \{\ldots,-1,0,1,\ldots\} \\ \mathbb Q& = \{\frac{p}{q}\vert p, q \in \mathbb Z, q\not = 0\} \end{aligned} \] 对于四则运算封闭的是 \(\mathbb Q, \mathbb R, \mathbb C\)

Dedekind分割

先定义一个分割

\(A, B\) 为有理数集合，满足：

  1. \(A, B \not = \emptyset, \mathbb Q = A \cup B, A \cap B = \emptyset\)
  2. \(\forall x\in A, y \in B, x < y\)

显然有四种情况

  1. \(A\) 中有最大值，\(B\) 中有最小值
  2. \(A\) 中无最大值，\(B\) 中有最小值
  3. \(A\) 中有最大值，\(B\) 中无最小值
  4. \(A\) 中无最大值，\(B\) 中无最小值

第一种情况不会出现，所以我们只讨论后面的三种。

Dedekind 分割就是 \(\{(A, B)是 \mathbb Q 的一个分割，且 A 中无最大值\}\)

每一个 Dedekind 分割唯一对应一个实数，无理数对应了第四种，有理数对应了第二种。

Dedekind 分割来定义无理数的运算

定义 \(x = (A_1, B_1), y = (A_2, B_2)\) ，若 \(A_1 \subset A_2\) 称 \(x\le y\)，若 \(A_1 \subset A_2\) 且 \(A_1 \not = A_2\) 称 \(x < y\)。

性质

  1. 反称性：\(x\le y\) 则 \(y\ge x\)
  2. 传递性：\(x\le y\) 且 \(y \le \mathbb Z\)，则 \(x \le \mathbb Z\)
  3. 完备性：\(x, y\in \mathbb R\)，则 \(x\le y\) 或 \(y\le x\)

四则运算定义

\(x + y = (A_3, B_3)\)，其中 \(A_3 = \{r \in \mathbb Q\vert \exists r_1 \in A_1, \exists r_2 \in A_2, \text{s.t.} r = r_1 + r_2\}\)， \(B_3 =\mathbb Q/A_3\)

负元 \(-x = (A_3, B_3)\)，其中 \(A_3 = \{r \in \mathbb Q\vert \exists r_1 \in B_1, r = -r_1, r_1 不是最小元\}\)， \(B_3 = \mathbb Q/A_3\)

则减法定义为 \(x - y = x + (-y)\)

乘法：

\(xy = (A_3, B_3)\)

  1. \(x\ge 0, y \ge 0, A_3 = \{r \in \mathbb Q\vert r < 0 或 \exists r_1 \in A_1, \exists r_2 \in A_2, \text{s.t.} r_1 \ge 0, r_2 \ge 0, r = r_1r_2\}, B = \mathbb Q / A_3\)
  2. \(x\ge 0, y < 0,xy = -(x(-y))\)
  3. \(x < 0, y < 0, xy = (-x)(-y)\)

逆元 \(A_3 = \{r \in Q\vert r\le 0 或 \exists r_1 \in B_1 , r_1 不是 B_1最小元,r=\frac{1}{r_1}\}, B = \mathbb Q / A_3\)

实数域范围内：

  1. 交换律
  2. 结合律
  3. 单位元
  4. 逆元
  5. 分配律

相容性条件

  1. 加法保序性 \(x\le y\) 则 \(x + z \le y + z\)
  2. 乘法保序性 \(x\le y, z \ge 0\), 则 \(xz\le yz\)

实数确界存在定理

定义 \(X \subset \mathbb R,x\not = \emptyset\)

  1. 若 \(C \in \mathbb R, \text{s.t.} \forall x \in X, x \le C\) 则称 \(C\) 是 \(X\) 的一个上界，下界同理
  2. 若 \(X\) 既有上界又有下界，那么称 \(X\) 是有界集
  3. 称 \(a\) 是 \(X\) 的一个上确界，若 \(\forall C\) 是 \(X\) 的上界，\(C \ge a\)，记 \(a = \sup X\) 上确界，下确界是 \(\inf X\)

定理：确界存在定理：有上界或者下界的非空子集，必有唯一的上确界或者下确界。

证明：非空集 \(X\) 有上界，定义集合 \(A_1 = \{ r \in \mathbb Q \vert \exists(A, B) \in X, \text{s.t.}r\in A\}\) 则 \(x_1 = (A_1, B_1) \in R\) 并且 \(\forall (A, B) \in X\) 我们有 \(A\subset A_1\) 故 \(x_1 = (A_1, B_1)\) 是上界，现在设 \(x_0 = (A_0, B_0)\) 是 \(X\) 的任一上界，根据定义，可以知道 \(A_1 \subset A_0, x_1 \le x_0\) 因此 \(x_1\) 是 \(X\) 的唯一上确界。

清华大学本科生日记

发表于 2023-09-13 更新于 2023-09-18

2023/9

16日

身体还不是很好，一直在低烧状态，15日晚上速通回家了，早上在和平区的某个早点铺子吃了早点，速通了天津大学的两个校区和南开大学的卫津路校区。

天津大学没有什么可以说的，主要说说看南开大学的体验，作为一个18年的天津人，我还是第一次进入南开大学，本来想找李欣然给我带进去，但是她回家了，于是找赵思远偷渡学生卡带我进去。

第一的印象还是有些破旧，果然百年的历史让这座学校变得有些沧桑，老校区还剩下一些基础科学的院系，还有经管学院，感觉很有实力。

化学系的楼非常高大，远大于物理和数学的总和。然后去看了省身数学研究所，气抖冷，为什么不叫省君数学研究所。

然后去拍了周恩来总理的石像，和南开中学里面的不同，那个是铜像，脸上总泛点绿，非常喜欢。

回家倒头便睡，醒来发现很难受，但是要回北京了，于是在高铁顶着感冒写出以上文字。

睡觉。

17日

今天上午参观了北大，王如意带我参观了北京大学，非常喜欢这种带有历史气息的学校，参观了国际数学中心非常有实力，见到了潘神，中途还和刘鸿玮碰到了一面。

中午把齐乐辰叫过来吃了个饭，感觉很好，羡慕图灵班。

下午准备各种东西，明天就要开始大学生活了。

成为了班级里面的组织委员。

明天的数学课不知道听不听得懂，早睡身体好。
